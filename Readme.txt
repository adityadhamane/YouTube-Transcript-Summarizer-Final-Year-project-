Steps for installaton:

1) Open the folder with the vs code:
2) In main.py file paste your openai api key and hugging face transformer key.
3) Go to the chrome and click on the 3 dots in the rightmost side of the chrome tab.
4) After clicking on 3 dots goto the 'More tools--->Extensions'.
5) In the Extension click on the 'load unpacked' and select the 'extension' file form the 'Project' folder.
6) After this step your extension will be appered in the chrome tab.
7) Just pin this extension to make popup.
8) Run the server of main.py file from 'py-summary' folder and index.js form 'server' file.
9) After this step your extension is ready yo use.
